## рҹҺЁ Design System

The entire visual identity runs on CSS custom properties:

```css
:root {
  /* в”Җв”Җв”Җ Gold Palette в”Җв”Җв”Җ */
  --gold:   #C9A84C;    /* Primary вҖ” buttons, prices, accents */
  --gold2:  #E8C97A;    /* Hover states */
  --gold3:  #8B6914;    /* Subtle borders */

  /* в”Җв”Җв”Җ Dark Palette в”Җв”Җв”Җ */
  --dark:   #0A0805;    /* True black вҖ” main bg */
  --dark2:  #120F0A;    /* Elevated surface вҖ” hero, about */
  --dark3:  #1C1610;    /* Card bg */
  --dark4:  #251E14;    /* Inset вҖ” gallery, booking */

  /* в”Җв”Җв”Җ Typography в”Җв”Җв”Җ */
  --text:   #F5EDD6;    /* Primary вҖ” warm ivory */
  --text2:  #B8A882;    /* Secondary вҖ” muted gold */
  --text3:  #7A6E5A;    /* Tertiary вҖ” deep muted */
}
```

**Fonts used:**
- `Playfair Display` вҖ” Logo, headings, prices (serif luxury)
- `Cormorant Garamond` вҖ” Body, labels, nav (refined readability)

### Adding a New Item

Open `index.html`, find the `menu` array in the `<script>` tag, and add:

```js
{
  id: 38,
  cat: 'hot',           // hot | cold | latte | frappe | special | food
  name: 'Your Item Name',
  desc: 'Short, enticing description',
  price: 'Rs 500',
  emoji: 'вҳ•',
  badge: 'New'          // Optional: Signature | Popular | New | Chef's Pick
}
```

Done. The UI updates automatically.
