## ◈ What is Cafe Eclipse?

> **Cafe Eclipse** is a modern, AI-assisted cafe web experience — designed to bring the warmth of a real café into the digital world. Built using **Google AI Studio** and deployed on **Vercel**, this project blends creative design with smart technology to deliver a seamless, immersive cafe interface.

Whether you're exploring the menu, soaking in the ambiance, or placing an order — Cafe Eclipse makes every interaction feel like a premium coffee moment. ☕🌑

## ◈ Getting Started

### Prerequisites
- A modern browser (Chrome, Firefox, Edge)
- Node.js (optional, for local dev)
- Git

### Clone & Run Locally

```bash
# 🔽 Clone the repository
git clone https://github.com/mawiya-47/Cafe-eclipse.git

# 📂 Navigate into the project
cd Cafe-eclipse

# 🌐 Open in browser
# Just open index.html — or use Live Server in VS Code!
```

### Deploy Your Own

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy in one command
vercel --prod
```
